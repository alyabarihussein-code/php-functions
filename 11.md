
## مع أمثلة عملية  

---

### **1. لماذا SQL Injection خطير حتى اليوم؟**
- **السبب**: لأنه يتيح للمهاجم تنفيذ أوامر SQL غير مصرح بها على قاعدة البيانات.
- **المثال**:  
  ```php
  // كود غير آمن
  $id = $_GET['id'];  // لو دخل المستخدم: 1; DROP TABLE users
  $sql = "SELECT * FROM users WHERE id = $id";
  ```
  النتيجة: يتم حذف جدول المستخدمين!
